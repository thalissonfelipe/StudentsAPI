# Students API

Simple Web API built with ASP .NET Core 3.1 and MongoDB for learning purposes.

## Dependencies

- .NET Core 3.1
- MongoDB

## Run

```bash
$ dotnet add package MongoDB.Driver --version 2.11.4
$ dotnet restore
$ dotnet build
$ dotnet run
```

## Simple Request

```bash
$ https://localhost:5001/api/students/
```
